# 文本对比工具 - Chrome 插件

一款简洁强大的文本对比工具,以 **并排双栏** 方式清晰展示两段文本之间的差异,支持按行、按单词、按字符三种对比粒度。

## ✨ 功能特性

- **并排双栏展示**:左右两栏同时显示原始与修改后的文本,差异一目了然
- **三种对比粒度**:按行 / 按单词 / 按字符,满足不同场景的对比需求
- **行间高亮**:不仅标记整行增删,还能在同一行内高亮具体发生变化的单词或字符
- **对比选项**:支持忽略大小写、忽略空白字符
- **内容持久化**:通过 `chrome.storage` 自动保存编辑内容和偏好设置,下次打开继续使用
- **便捷操作**:一键交换左右内容、一键清空、统计新增/删除行数
- **快捷键**:`Ctrl/Cmd + Enter` 快速开始对比,`Esc` 返回编辑

## 🚀 加载到 Chrome

1. 打开 Chrome,访问 `chrome://extensions/`
2. 开启右上角的 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目的根目录 `text-diff-extension/`
5. 工具栏上会出现一个蓝色 ⇌ 图标,点击即可在新标签页中打开对比工具

## 🎯 使用方法

1. 点击工具栏中的插件图标,在新标签页中打开对比工具
2. 在左侧输入/粘贴 **原始文本**,在右侧输入/粘贴 **修改后文本**
3. (可选) 在顶部工具栏选择对比粒度、是否忽略大小写或空白
4. 点击 **开始对比** 按钮,或按下 `Ctrl/Cmd + Enter`
5. 查看并排高亮结果:
   - 🟥 红色 = 删除的内容(仅在左侧)
   - 🟩 绿色 = 新增的内容(仅在右侧)
   - 同一行内的差异会以更深的色块精确高亮

## ⌨️ 快捷键

| 快捷键 | 功能 |
|---|---|
| `Ctrl/Cmd + Enter` | 开始对比 |
| `Esc` | 返回编辑视图 |
